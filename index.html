Plan:

Laplace$\to$Undetermined coefficients$\to$inv. Laplace

Things behave nicely, so everything can be done analytically up to the inverse Laplace transform.

\[ EI \partial_x^4 y +m\partial_t^2 y + C(x)\partial_t y + k(x)y = P\delta(x-vt) \]

Apply the Laplace transform, $\hat y = \int_0^\infty y e^{-st} dt$:

$\downarrow$

\[EI \partial x^4 \hat y + [ms^2 + C(x)s + k(x)]\hat y = \begin{cases} P \delta(x) \quad v=0,\\ \frac{P}{|v|}e^{-sx/v}\quad v\ne 0.\end{cases} \]
Now this is an ordinary differential equation which we can solve in $x$. I am going to focus on the case where $v=0$ (partially because otherwise the undetermined coefficients part goes to zero).

System with 4 splits in beam:

\[\begin{align*} \begin{aligned} -c_1 + c_2 + c_3 -c_4 = 0,\\ -r_1c_1 + r_2 c_2 + r_3 c_3 - r_4 c_4 = 0,\\ -r_1^2 c_1 + r_2^2 c_2 + r_3^2 c_3 - r_4^2 c_4 = 0,\\ -r_1^3 c_1 + r_2^3 c_2 + r_3^3 c_3 - r_4^3 c_4 = \frac{P}{EIs}.\\ \end{aligned} \end{align*}\]

For each homogeneous section we have to invert a Laplace-space solution of the form \[ \small{ \bar{y}(x,s) = c_1(s)e^{r_1(s)x} + c_2(s)e^{r_2(s)x} + c_2(s)e^{r_3(s)x} + c_4(s)e^{r_4(s)x}, } \] with \[\scriptsize{ r^4 + \frac{ms^2 + Cs + k}{EI} = 0.}\]
The $c_j$ are found by solving a linear system. Inverting analytically isn't really an option, so we resort to numerical techniques.
